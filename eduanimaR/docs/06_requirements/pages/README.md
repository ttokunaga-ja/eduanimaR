# Page Requirements（Index）

このフォルダは、各ページ（route）の要件を管理します。

- テンプレ：`PAGE_REQUIREMENTS_TEMPLATE.md`
- 新規作成時はテンプレを複製して埋めます

推奨命名：`P_<id>_<PageName>_REQUIREMENTS.md`

## Pages（SSOT）

- `P_1_ChatWorkspace_REQUIREMENTS.md`（ハイブリッド・タブ・レイアウトのアプリシェル）
- `P_2_FileManagementCenter_REQUIREMENTS.md`（全画面：資料管理センター）
- `P_3_HistoryArchive_REQUIREMENTS.md`（全画面：ラーニング・アーカイブ）
